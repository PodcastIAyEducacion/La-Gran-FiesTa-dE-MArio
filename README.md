# La-Gran-FiesTa-dE-MArio
Recurso HTML5 interactivo para el desarrollo de la lectoescritura en alumnado con TEA (7 años). 6 fases narrativas con el universo de Mario Bros. Incluye lectura con iluminación de palabras, comodines con mediación docente, actividades en papel y diploma imprimible. El docente tiene un papel activo en cada fase.
